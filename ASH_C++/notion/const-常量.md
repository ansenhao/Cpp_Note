内容可修改  指针不可以修改 ^pWyydO8x

例： ^slMQ5TDk

当行参是 int double 这些时，        它们只是实参的副本，修改它们不影响实参
当行参是 int* double* 这类指针时，  它们的在形参中修改会一并修改掉实参
所以对有指针类型的函数，常加const ^R7VX4jVV

p2是变量，允许修改，但是不会使用drivel这个函数修改，所以可行 ^vUWtPuBz

总结：p2允许被修改，放入参数列表是const 和非const 都可以

const p1 不允许被修改，放入参数列表是const的可以
                       放入参数列表是非const的不可以 ^F3bkzdxp
